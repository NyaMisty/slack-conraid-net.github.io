Repository contains 64bit packages for Slackware Current
Each directory contain the binary packages, ready for installation using 'installpkg' or 'upgradepkg'. You'll find *.txz (the package itself), *.txt (the package description) and *.md5 (containing the md5 checksum of the package) files in each directory.

-extra repository contains packages that need external dependencies, not present in my repository, like kde5 for example.

SLACKBUILD and other files needed for compiling packages can be found on GitHub
