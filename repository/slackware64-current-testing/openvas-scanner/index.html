openvas-scanner (OpenVAS remote network security scanner)

This is the Open Vulnerability Assessment System (OpenVAS) Scanner of
the Greenbone Vulnerability Management (GVM) Solution.
It is used for the Greenbone Security Manager appliances and is a
full-featured scan engine that executes a continuously updated and
extended feed of Network Vulnerability Tests (NVTs).

HOME: http://www.openvas.org

greenbone-security-assistant

This is the UI the Greenbone Vulnerability Manager

###### Installation Instructions ######

You must install all gvm packages (gsa, gvm-libs, gvm-tools, gvmd,
openvas, ospd and python-gvm) and related dependencies
(redis, libmicrohttpd, node, paramiko, lxml, defusedxml).

Then you can follow the instructions in the gsa package.
