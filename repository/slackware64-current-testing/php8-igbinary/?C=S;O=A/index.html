php-igbinary (php igbinary extension)

Igbinary is a drop in replacement for the standard php serializer.
Instead of the time and space consuming textual representation used
by PHP's serialize, igbinary stores php data structures in a compact
binary form.

HOME: https://github.com/igbinary/igbinary/

REQUIRES: php8
