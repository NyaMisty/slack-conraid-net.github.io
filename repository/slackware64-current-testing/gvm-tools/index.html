gvm-tools (Greenbone Vulnerability Management Tools)

GVM-Tools is a collection of tools that help with remote controlling
a Greenbone Security Manager (GSM) appliance and its underlying
Grennbone Vulnerability Manager (GVM). The tools essentially aid
accessing the communication protocols GMP (Greenbone Management
Protocol) and OSP (Open Scanner Protocol).

Previously known as OpenVas-Tools

HOME: https://community.greenbone.net

This is the UI the Greenbone Vulnerability Manager

###### Installation Instructions ######

You must install all gvm packages (gsa, gvm-libs, gvm-tools, gvmd,
openvas-scanner, ospd and python-gvm) and related dependencies
(redis, libmicrohttpd, node, paramiko, lxml, defusedxml).

Then you can follow the instructions in the gsa package.
