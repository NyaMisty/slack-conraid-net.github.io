dpkg (Debian package management system)

This package provides the development tools (including dpkg-source)
required to unpack, build and upload Debian source packages.

In addition, this package also provides the low-level infrastructure
for handling the installation and removal of Debian packages in 
Slackware system, but do not do it. It can have DISASTROUS effects.

HOME: http://wiki.debian.org/Teams/Dpkg
