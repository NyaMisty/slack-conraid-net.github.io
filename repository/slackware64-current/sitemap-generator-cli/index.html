sitemap-generator-cli (Create xml sitemaps from the command line)

Generates a sitemap by crawling your site. Uses streams to
efficiently write the sitemap to your drive. Is cappable of creating
multiple sitemaps if threshold is reached. Respects robots.txt
and meta tags.
