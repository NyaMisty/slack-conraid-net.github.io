md5deep (Advanced checksum hashing tool)

md5deep is a set of programs to compute MD5, SHA-1, SHA-256, Tiger,
or Whirlpool message digests on an arbitrary number of files. 
md5deep is similar to the md5sum program found in the GNU Coreutils
package, but has the following additional features
