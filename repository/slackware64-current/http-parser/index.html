http-parser (http request/response parser for c )

http-parser for HTTP messages written in C. It parses both requests
and responses. The parser is designed to be used in performance HTTP
applications.  It does not make any syscalls nor allocations, 
it does not buffer data, it can be interrupted at anytime.

HOME: https://github.com/nodejs/http-parser
