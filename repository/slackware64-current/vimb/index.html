vimb (keyboard-driven, vim-like web browser)

Vimb is a webkit2gtk-based browser inspired by Pentadactyl and
Vimprobable. The goal of Vimb is to build a completely
keyboard-driven, efficient and pleasurable browsing experience with
low memory and CPU usage that is intuitive to use for Vim users.

HOME: https://fanglingsu.github.io/vimb/

======================================================================
REQUIRES: webkitgtk
