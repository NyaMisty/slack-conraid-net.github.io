cmocka (Unit testing framework for C)

CMocka is an elegant unit testing framework for C with support for
mock objects. It provides a powerful testing framework for C, on
on different platforms and operating systems, which only requires the
standard C library.

It includes support for mock objects, a fully documented API, test
fixtures, exception handling for singlas, testing of memory leaks and
buffer overflows and underflows.

HOME: https://cmocka.org/
