fwbuilder (Firewall creation utility)

Firewall Builder consists of a GUI and set of policy compilers for 
various firewall platforms. It helps users maintain a database of 
objects and allows policy editing using simple drag-and-drop
operations. Firewall Builder currently supports iptables, ipfilter, 
OpenBSD PF and Cisco PIX.

HOME: http://www.fwbuilder.org
