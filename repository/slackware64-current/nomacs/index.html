nomacs (Qt image viewer)

nomacs is a free image viewer for windows and linux systems.
nomacs is small,fast and able to handle the most common image formats.
Additionally it is possible to synchronize multiple viewers.

HOME: http://www.nomacs.org/

======================================================================
REQUIRES: qt-heif-image-plugin qt-avif-image-plugin
