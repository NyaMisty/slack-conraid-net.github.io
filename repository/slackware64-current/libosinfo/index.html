libosinfo (operating systems library)

GObject based library API for managing information about operating
systems, hypervisors and the (virtual) hardware devices they can
support. It includes a database containing device metadata and
provides APIs to match/identify optimal devices for deploying
an operating system on a hypervisor.

HOME: https://releases.pagure.org/libosinfo

REQUIRES: osinfo-db
