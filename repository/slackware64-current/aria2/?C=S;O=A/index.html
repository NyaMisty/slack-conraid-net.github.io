Aria2 (CLI Metalink/BiTorrent client)

aria2 is a utility for downloading files. The supported protocols
are HTTP(S), FTP, BitTorrent, Metalink. It can download a file
from multiple sources/protocols and tries to utilize maximum
download bandwidth.
