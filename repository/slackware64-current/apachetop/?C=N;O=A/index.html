apachetop (real time apache monitoring tool)

A curses-based top-like display for Apache information, including
requests per second, bytes per second, most popular URLs, etc.
ApacheTop watches a logfile generated by Apache (in standard common
or combined logformat, and generates human-parsable output in
realtime.

HOME: https://github.com/tessus/apachetop
