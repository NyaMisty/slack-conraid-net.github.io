dos2unix (DOS/MAC to UNIX text file format converter)

Convert text files with DOS line endings to Unix line endings and 
vice versa.
This is an update of Benjamin Lin's implementations of dos2unix and
unix2dos. Benjamin Lin's implementations of dos2unix and unix2dos are
part of all RedHat based Linux distributions and others.

HOME: http://www.xs4all.nl/~waterlan/dos2unix.html
