libcli (cisco style telnet commandline interface library)

Libcli provides a shared library for including a Cisco-like command-
line interface into other software.
It's a telnet interface which supports command-line editing, history,
authentication and callbacks for a user-definable function tree.

HOME: https://github.com/dparrish/libcli
