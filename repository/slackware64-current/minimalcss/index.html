minimalcss (Extract the minimal CSS used in a set of URLs)

A Node library to extract the minimal CSS used in a set of URLs with
puppeteer. Used to find what minimal CSS is needed to render on first
load, even with document.onload executed.

HOME: https://minimalcss.app/
