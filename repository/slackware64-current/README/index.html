Slackers (Slackware Current Repository by Conraid) ================================================================================ Repository contains 64bit packages for Slackware Current The package directory contains the binary packages, ready for installation using 'installpkg' or 'upgradepkg'. You'll find *.txz (the package itself), *.txt (the package description) and *.md5 (containing the md5 checksum of the package) files in each directory. SLACKBUILD and other files needed for compiling packages can be found on github https://github.com/conraid/SlackBuilds If you have any questions or requests, you can contact me at slackers.it (at) gmail (dot) com Packaged by Corrado Franco (https://conraid.net) Support me https://paypal.me/conraid
