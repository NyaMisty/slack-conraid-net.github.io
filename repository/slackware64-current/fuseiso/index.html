fuseiso (FUSE module to mount ISO filesystem images)

FuseISO is a FUSE module to mount ISO filesystem images
(.iso, .nrg, .bin, .mdf and .img files). It currently support plain 
ISO9660 Level 1 and 2, Rock Ridge, Joliet, and zisofs.

HOME: https://sourceforge.net/projects/fuseiso/
