lgi (Lua binding to GObject libraries using GObject-Introspection)

LGI is gobject-introspection based dynamic Lua binding to GObject
based libraries. It allows using GObject-based libraries directly
from Lua.

HOME: https://github.com/lgi-devs/lgi

======================================================================
REQUIRES: lua lua53 lua52 lua51
