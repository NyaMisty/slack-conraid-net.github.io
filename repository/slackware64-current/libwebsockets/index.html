libwebsockets (C library for websocket clients and servers)

Libwebsockets is a simple-to-use, MIT-license, pure C library
providing client and server for http/1, http/2, websockets, MQTT and
other protocols in a security-minded, lightweight, configurable,
scalable and flexible way. It's easy to build and cross-build via
cmake and is suitable for tasks from embedded RTOS through mass cloud
serving.

HOME: https://github.com/warmcat/libwebsockets/
