kirigami-addons (Add-ons for the Kirigami framework)

A set of "widgets" i.e visual end user components along with a code to
support them. Components are usable by both touch and desktop
experiences providing a native experience on both, and look native
with any QQC2 style (qqc2-desktop-theme, Material or Plasma)

HOME: https://invent.kde.org/libraries/kirigami-addons
