instant-markdown (Instant Markdown previews from VIm)

instant-markdown is a small Node.js server that enables instant
compilation and previewing of Markup files. A plugin can easily be
written for any text editor to interface with it. 

HOME: https://github.com/Youthink/instant-markdown
