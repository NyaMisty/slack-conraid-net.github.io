libtermkey (keyboard entry processing library)

This library allows easy processing of keyboard entry from 
terminal-based programs. It handles all the necessary logic
to recognise special keys, UTF-8 combining, and so on,
with a simple interface.

HOME: https://github.com/neovim/libtermkey
