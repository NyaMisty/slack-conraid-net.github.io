pynvim (python client and plugin host for neovim)

Pynvim implements support for python plugins in Nvim (neovim).
It also works as a library for connecting to and scripting Nvim
processes through its msgpack-rpc API.

HOME: https://github.com/neovim/pynvim

======================================================================
REQUIRES: greenlet msgpack-python neovim
