python-pikepdf (Read and write PDFs with Python, powered by qpdf)

pikepdf is a Python library allowing creation, manipulation and
repair of PDFs. It provides a Pythonic wrapper around the C++ PDF
content transformation library, QPDF.

HOME: https://pikepdf.readthedocs.io/

REQUIRES: lxml pybind11
