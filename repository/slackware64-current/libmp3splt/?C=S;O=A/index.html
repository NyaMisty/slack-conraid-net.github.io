libmp3splt (library from the mp3splt project)

mp3splt is a free utility to split mp3/ogg files (without decoding),
selecting begin/end time; if file is an album, you can get splitpoints
automatically from internet or a local cue/cddb file.
It splits also Mp3Wrap and AlbumWrap archives.

HOME: http://mp3splt.sourceforge.net
