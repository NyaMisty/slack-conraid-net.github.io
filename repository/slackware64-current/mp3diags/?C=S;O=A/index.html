MP3 Diags 

MP3 Diags is a GUI-based application that allows end-users to 
identify issues with their MP3 files, fix some of the issues and 
make other changes, like adding track information. 
It also lets you "look inside" an MP3 file. 



HOME: http://mp3diags.sourceforge.net/
