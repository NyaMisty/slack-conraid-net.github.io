gimp-separate+ (CMYK support for GIMP)

separate+ is a CMYK plugin for the GNU Image Manipulation Program
which brings improved ICC color management, soft-proof color, color
profile switching, and duotone support to your workflow...plus,
obviously, the ability to separate colors from RGB images.

HOME: https://osdn.net/projects/separate-plus
