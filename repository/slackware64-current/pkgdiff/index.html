pkgdiff (A tool for visualizing changes in Linux software packages)

Package Changes Analyzer (pkgdiff) is a tool for visualizing changes
in Linux software packages (RPM, DEB, TAR.GZ, etc).
The tool is intended for Linux maintainers who are interested in
ensuring compatibility of old and new versions of packages.

HOME: https://lvc.github.io/pkgdiff/
