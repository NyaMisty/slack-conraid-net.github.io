clamav (open source antivirus engine)

Clam AntiVirus is a GPL anti-virus toolkit for UNIX. The main purpose
of this software is the integration with mail servers (attachment
scanning). The package provides a flexible and scalable multi-threaded
daemon, a command line scanner, and a tool for automatic updating via
Internet.  Most importantly, the virus database is kept up to date.

HOME: http://www.clamav.net

======================================================================
NOTE: 

Run freshclam to download some virus definitions before starting clamav.

To build Clamav with this SlackBuild is required libmilter package.
