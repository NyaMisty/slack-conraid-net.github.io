tig (git ncurses interface)

Tig is an ncurses-based text-mode interface for git.
It functions mainly as a Git repository browser, but can also assist
in staging changes for commit at chunk level and act as a pager
for output from various Git commands.

HOME: http://jonas.nitro.dk/tig/
