ISO Master (easy to use, graphical CD image editor)

ISO Master is an open-source, easy to use, graphical CD image
editor for Linux and BSD. Basically you can use this program to
extract files from an ISO, add files to an ISO, and create
bootable ISOs - all in a graphical user interface. It can open
both ISO and NRG files but can only save as ISO.

HOME: http://www.littlesvr.ca/isomaster/
