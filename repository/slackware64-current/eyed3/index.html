eyed3 (Python module and commandline program for processing ID3 tags)

eyeD3 is a Python tool for working with audio files, specifically
MP3 files containing ID3 metadata (i.e. song info).
It provides a command-line tool (eyeD3) and a Python library
(import eyed3) that can be used to write your own applications or
yed3. plugins that are callable from the command-line tool.

HOME: http://eyed3.readthedocs.io/en/latest/
