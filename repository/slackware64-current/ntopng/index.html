ntopng (next generation network top)

ntopng is a tool for both Unix and Win32 that shows the network 
usage, similar to what the popular top Unix command does. 
It sports a web interface for accessing accounting data and includes
support for popular tools/protocols as well DPI and host categorisation.

HOME: https://www.ntop.org
