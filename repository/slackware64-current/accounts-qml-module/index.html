accounts-qml-module (QML bindings for libaccounts-qt + libsignon-qt)

This QML module provides an API to manage the user's online accounts
and get their authentication data.
It's a tiny wrapper around the Qt-based APIs of libaccounts-qt
and libsignon-qt.

HOME: https://gitlab.com/accounts-sso/accounts-qml-module

======================================================================
REQUIRES: libaccounts-qt signond
