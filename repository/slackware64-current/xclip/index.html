xclip (Command line interface to the X11 clipboard)

xclip is a command line interface to the X11 clipboard. It can also
be used for copying files, as an alternative to sftp/scp, thus
avoiding password prompts when X11 forwarding has already been setup.

HOME: https://sourceforge.net/projects/xclip/
