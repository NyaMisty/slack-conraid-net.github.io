lterm ( free terminal and multiprotocol client for Linux)

lterm is a vte-based terminal emulator for GNU/Linux systems. It is
mainly used as SSH/Telnet client but can be very easily configured
to use any other protocol. Furthermore it can be a usual terminal
on local host.
Includes some features in order to ease user's work, avoid
repetitive tasks and improve speed in operations. 

HOME: http://lterm.sourceforge.net
