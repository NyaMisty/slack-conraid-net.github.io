sleuthkit (tsk)

The Sleuth Kit (TSK) is a collection of UNIX-based command line tools
that allow you to investigate a computer. The current focus of the 
tools is the file and volume systems and TSK supports FAT, Ext2/3, 
NTFS, UFS, and ISO 9660 file systems

HOME: http://www.sleuthkit.org/sleuthkit/index.php
