soupsieve (A modern CSS selector implementation for BeautifulSoup)

Soup Sieve is a CSS selector library designed to be used with
Beautiful Soup 4. It aims to provide selecting, matching, and
filtering using modern CSS selectors. Soup Sieve currently provides
selectors from the CSS level 1 specifications up through the latest
CSS level 4 drafts (though some are not yet implemented).

HOME: https://github.com/facelessuser/soupsieve
