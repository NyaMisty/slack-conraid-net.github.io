Slackware Current Repository by Conraid ====================================================================== unix-privesc-check (Shell script to check for privilege escalation) Unix-privesc-checker is a script that runs on Unix systems. It tries to find misconfigurations that could allow local unprivileged users to escalate privileges to other users or to access local apps (e.g. databases).
