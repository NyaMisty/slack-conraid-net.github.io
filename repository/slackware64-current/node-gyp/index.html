node-gyp (Node.js native addon build tool)

node-gyp is a cross-platform command-line tool written in Node.js
for compiling native addon modules for Node.js. It bundles the gyp
project used by the Chromium team and takes away the pain of dealing
with the various differences in build platforms. 

HOME: https://github.com/nodejs/node-gyp/

REQUIRES: node-semver
