python-polib (A library to manipulate gettext files)

polib is a library to manipulate, create, modify gettext files (pot,
po and mo files). You can load existing files, iterate through it’s
entries, add, modify entries, comments or metadata, etc… or create
new po files from scratch.

HOME: http://bitbucket.org/izi/polib/
