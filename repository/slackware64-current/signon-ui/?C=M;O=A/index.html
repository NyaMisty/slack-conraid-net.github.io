signon-ui (Online Accounts Sign-on UI)

Sign-on UI is the component responsible for handling the user
interactions which can happen during the login process of an online
account.
It can show password dialogs and dialogs with embedded web pages.

HOME: https://launchpad.net/signon-ui

REQUIRES: libaccounts-qt signond accounts-qml-module
