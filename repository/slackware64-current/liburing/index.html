liburing (Linux-native io_uring I/O access library)

This is the io_uring library, liburing. liburing provides helpers to
setup and teardown io_uring instances, and also a simplified interface
for applications that don't need (or want) to deal with the full
kernel side implementation.

HOME: https://github.com/axboe/liburing/
