john (John The Ripper)

John the Ripper is a fast password cracker, currently available for 
many flavors of Unix. Its primary purpose is to detect weak Unix 
passwords. It supports several crypt(3) password hash types which are
most commonly found on various Unix flavors, as well as Kerberos AFS
and Windows NT/2000/XP LM hashes. 

HOME: http://www.openwall.com/john/

======================================================================
REQUIRES: lua openmpi perl-digest-md4
