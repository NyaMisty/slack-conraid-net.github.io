gnuit (GNU Interactive Tools)

gnuit is a set of interactive text-mode tools, closely integrated
with the shell. It contains an extensible file system browser, an
ascii/hex file viewer, a process viewer/killer and some other related
utilities and shell scripts. 

HOME: http://www.gnu.org/software/gnuit/
