Slackware Current Repository by Conraid ====================================================================== smmap2 (A sliding memory map manager) Smmap wraps an interface around mmap and tracks the mapped files as well as the amount of clients who use it. If the system runs out of resources, or if a memory limit is reached, it will automatically unload unused maps to allow continued operation.
