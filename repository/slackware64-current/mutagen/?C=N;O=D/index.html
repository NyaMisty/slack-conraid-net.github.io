mutagen (Python 2/3 module for handling audio metadata)

Mutagen is a Python module to handle audio metadata. It supports ASF,
FLAC, MP4, Monkey's Audio, MP3, Musepack, Ogg Opus, Ogg FLAC,
Ogg Speex, Ogg Theora, Ogg Vorbis, True Audio, WavPack, OptimFROG,
and AIFF audio files. All versions of ID3v2 are supported, and all
standard ID3v2.4 frames are parsed. It can read Xing headers to
accurately calculate the bitrate and length of MP3s.

HOME: https://mutagen.readthedocs.io/
