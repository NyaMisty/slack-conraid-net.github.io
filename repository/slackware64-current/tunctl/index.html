tunctl  (tool for controlling the TUN/TAP driver in Linux)

tunctl is used to set up and maintain persistent TUN/TAP
network interfaces, enabling user applications to simulate
network traffic. Such interfaces is useful for VPN software,
virtualization, emulation, simulation, and a number of other
applications.

HOME: http://tunctl.sourceforge.net/
