mysqltuner (MySQL Tuning Script)

MySQLTuner is a script written in Perl that allows you to review a
MySQL installation quickly and make adjustments to increase
performance and stability. The current configuration variables and
status data is retrieved and presented in a brief format along with
some basic performance suggestions.

HOME: http://mysqltuner.com
