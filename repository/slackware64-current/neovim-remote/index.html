neovim-remote (Support --remote and friends for Neovim)

nvr provides an executable called nvr which solves these cases:
  - Controlling nvim processes from the shell.
    E.g. opening files in another terminal window.
  - Opening files from within :terminal without starting a nested
    nvim process.

HOME: https://github.com/mhinz/neovim-remote

REQUIRES: pynvim
