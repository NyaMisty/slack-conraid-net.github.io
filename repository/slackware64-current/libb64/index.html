libb64 (C routines for base64 encoding/decoding)

libb64 is a library of ANSI C routines for fast encoding/decoding data
into and from a base64-encoded format. C++ wrappers are included, as
well as the source code for standalone encoding and decoding
executables.

HOME: https://github.com/libb64/libb64
