phpspy (Low-overhead sampling profiler for PHP 7)

phpspy is a low-overhead sampling profiler for PHP 7. For now,
it works with Linux 3.2+ x86_64 non-ZTS PHP 7.0+ with CLI, Apache,
and FPM SAPIs.
