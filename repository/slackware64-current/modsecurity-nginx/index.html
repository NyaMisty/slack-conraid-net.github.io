ModSecurity (ModSecurity v3 Nginx Connector)

The ModSecurity-nginx connector is the connection point between
Nginx and libmodsecurity (ModSecurity v3). Said another way, this
project provides a communication channel between Nginx and
libmodsecurity. This connector is required to use LibModSecurity
with Nginx.

HOME: https://www.modsecurity.org/

======================================================================
NOTE: 

FOR TESTING AND DEVELOPMENT ONLY

You can get started with ModSecurity by adding the following lines to the
appropriate sections of Apache's main configuration file (httpd.conf):

Include /etc/httpd/mod_security3.conf


======================
REQUIRES: libmodsecurity nginx
