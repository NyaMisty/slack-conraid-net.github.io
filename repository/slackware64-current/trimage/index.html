trimage (A cross-platform tool for optimizing PNG and JPG files)

Trimage is a cross-platform GUI and command-line interface to optimize
image files via advpng, jpegoptim, optipng and pngcrush depending on
the filetype (currently, PNG and JPG files are supported).

HOME: http://trimage.org/

======================================================================
REQUIRES: advancecomp jpegoptim optipng pngcrush
