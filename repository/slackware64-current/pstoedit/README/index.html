Slackware Current Repository by Conraid ====================================================================== pstoedit (vector graphics converter for ps and pdf files) pstoedit converts PostScript and PDF files to various vector graphic formats. The resulting files can be edited or imported into various drawing packages. Output formats include: AI, ASY, CGM, DXF, EMF, FIG, GMFA, GMFI, GSCHEM, HPGL, IDRAW, KIL, LATEX2E, LWO, MIF, MNA, MPOST, NOIXML, PCB, PCL, RIB, RPL, RTF, SK, SVG, SVM, SWF, TGIF, XAML, WMF, and variants of these. Output is also possible to various text formats, Java, or TK code. HOME: http://www.pstoedit.net/ ======================================================================
