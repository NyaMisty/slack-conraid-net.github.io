ms-sys: ms-sys (Program for writing Microsoft compatible boot records)
ms-sys:
ms-sys: This is a Linux program for writing Microsoft compatible boot records.
ms-sys: The program does the same as Microsoft "fdisk /mbr" to a hard disk or
ms-sys: "sys d:" to a floppy or FAT partition except that it does not copy any
ms-sys: system files, only the boot record is written.
