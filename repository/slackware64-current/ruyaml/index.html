ruyaml (YAML parser/emitter for Python)

ruamel.yaml is a YAML parser/emitter that supports roundtrip
preservation of comments, seq/map flow style, and map key order
ruyaml package is a fork of ruamel.yaml aimed to made in order to
secure the future of the library, mainly by having a pool of
maintainers.
