======================================================================

vimtips (aggregate and show vim tips)

Vim Tips is a project to aggregate and show vim tips from different
sources. A small daemon waits for screen unlocks and automatically
starts the vim tips gui if it is the first unlock of the day.

HOME: https://github.com/IngoHeimbach/vimtips
