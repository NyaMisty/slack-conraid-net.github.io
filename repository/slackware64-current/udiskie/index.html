udiskie (Removable disk automounter for udisks)

udiskie is a simple daemon that uses UDisks to automatically mount
removable storage devices. This daemon comes with optional mount
notifications and gtk tray icon. It also provides a user level CLI
for mount and unmount operations.

HOME: https://github.com/coldfix/udiskie
