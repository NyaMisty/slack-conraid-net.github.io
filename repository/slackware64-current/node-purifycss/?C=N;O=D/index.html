purifycss (Remove unused CSS. Also works with single-page apps)

A function that takes content (HTML/JS/PHP/etc) and CSS, and returns
urifycss: only the used CSS. PurifyCSS does not modify the original CSS files.
urifycss: You can write to a new file, like minification.
urifycss: If your application is using a CSS framework, this is especially
urifycss: useful as many selectors are often unused.

HOME: https://github.com/purifycss/purifycss/
