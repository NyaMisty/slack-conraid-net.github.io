simplejson (extensible JSON encoder and decoder for Python)

Simplejson is a simple, fast, complete, correct and extensible JSON
encoder and decoder for Python.
It is pure Python code with no dependencies, but includes an optional
C extension for a serious speed boost.

HOME: https://github.com/simplejson/simplejson
