maldet (Linux Malware Detection (LMD))

Linux Malware Detect (LMD) is a malware scanner for Linux released
under the GNU GPLv2 license, that is designed around the threats 
faced in shared hosted environments. It uses threat data from network
edge intrusion detection systems to extract malware that is actively
being used in attacks and generates signatures for detection.

HOME: https://github.com/rfxn/linux-malware-detect
