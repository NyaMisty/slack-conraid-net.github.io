Slackware Current Repository by Conraid ====================================================================== mps-youtube (Terminal based YouTube player and downloader) This project is based on mps, a terminal based program to search, stream and download music. This implementation uses YouTube as a source of content and can play and download video as well as audio. The pafy library handles interfacing with YouTube. HOME: https://github.com/mps-youtube/mps-youtube ====================================================================== REQUIRES: python-pafy
