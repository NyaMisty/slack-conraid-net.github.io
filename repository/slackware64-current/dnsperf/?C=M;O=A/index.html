dnsperf (Tools that measure performance of authoritative DNS)

dnsperf and resperf are free tools that make it simple to gather
accurate latency and throughput metrics for Domain Name Service (DNS).
These tools are easy-to-use and simulate typical Internet, so network
operators can benchmark their naming and addressing infrastructure
and plan for upgrades

HOME: https://www.dns-oarc.net/tools/dnsperf
