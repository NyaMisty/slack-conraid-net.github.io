Slackware Current Repository by Conraid ====================================================================== libewf Libewf is a library for support of the Expert Witness Compression Format (EWF), it support both the SMART format (EWF-S01) and the EnCase format (EWF-E01). Libewf allows you to read and write media information within the EWF files. HOME: https://github.com/libyal/libewf ======================================================================
