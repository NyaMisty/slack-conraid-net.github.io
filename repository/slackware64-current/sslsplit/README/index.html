Slackware Current Repository by Conraid ====================================================================== sslsplit (transparent and scalable SSL/TLS interception) SSLsplit is a tool for man-in-the-middle attacks against SSL/TLS encrypted network connections. Connections are transparently intercepted through a network address translation engine and redirected to SSLsplit.
