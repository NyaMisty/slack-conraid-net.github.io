qt-heif-image-plugin (Qt plugin for HEIF images)

This is a Qt image plugin for HEIF images, wrapping libheif.
It enables existing Qt applications to open and save .heic files,
such as those from Apple devices.

HOME: https://github.com/jakar/qt-heif-image-plugin

======================================================================
REQUIRES: libheif
