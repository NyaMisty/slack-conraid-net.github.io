libdbi (Database Independent Abstraction Layer for C)

libdbi implements a database-independent abstraction layer in C,
similar to the DBI/DBD layer in Perl. Writing one generic set
set of code, programmers can leverage the power of multiple
databases and multiple simultaneous database connections by using
this framework.

HOME: http://libdbi.sourceforge.net/
