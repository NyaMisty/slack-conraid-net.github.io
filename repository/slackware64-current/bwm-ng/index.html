Bwm-ng (Realtime bandwidth monitor using curses)

Bandwidth Monitor NG is a small and simple curses-based bandwidth
monitor for Linux or other systems with /proc/net/dev or net-tools
netstat. You can choose whether to show all interfaces or only those
interfaces that are up.

HOME: http://www.gropp.org/?id=projects&sub=bwm-ng
