libircclient (a small client-server IRC protocol library)

libircclient is a small but powerful library, which implements
client-server IRC protocol. It is designed to be small, fast,
portable and compatible to RFC standards, and most IRC clients.

HOME: http://www.ulduzsoft.com/linux/libircclient/
