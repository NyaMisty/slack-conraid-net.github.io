Slackware Current Repository by Conraid ====================================================================== pixiewps (An offline WPS brute-force utility) Pixiewps is a tool written in C used to bruteforce offline the WPS pin exploiting the low or non-existing entropy of some Access Points, the so-called "pixie dust attack" discovered by Dominique Bongard in summer 2014. It is meant for educational purposes only. HOME: https://github.com/wiire-a/pixiewps ======================================================================
