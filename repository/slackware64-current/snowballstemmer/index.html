snowballstemmer (16 stemmer algorithms for python)

snowballstemmer is an extension to Flask that adds i18n and l10n
support to any Flask application with the help of babel, pytz and
speaklater.
It has builtin support for date formatting with timezone support as
well as a very simple and friendly interface to gettext translations.
