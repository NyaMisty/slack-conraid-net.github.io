urlscan (Mutt and terminal url selector)

Urlscan is a small program that is designed to integrate with the 
"mutt" mailreader to allow you to easily launch a Web browser for 
URLs contained in email messages. 
It is a replacement for the "urlview" program.

HOME: https://github.com/firecat53/urlscan
